# Audio files for Lead The Dragon website

Place your files here with these exact names:

| File | Purpose |
|------|---------|
| `bg-music.mp3` | Background music (loops after intro) |
| `enter.mp3` | Sound when user clicks "Enter" on intro |
| `click.mp3` | Button click SFX (nav, links, toast) |
| `hover.mp3` | Optional hover SFX on buttons/links |

To change the track name shown in the toast, edit `TRACK_TITLE` in `src/audio/paths.ts`.

Supported format: MP3 (recommended for browser compatibility).
